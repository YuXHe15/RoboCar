# LiDAR SSMR 

A LiDAR SSMR system built from scratch. 
![SSMR img](/pics/IMG_2131.jpeg)

## Subsystem
+ Motor Control
+ Kinetic System[^1]
+ Navigator
+ Command Pool
+ Backend (Communication based on RestAPI)
